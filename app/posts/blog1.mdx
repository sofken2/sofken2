## 作り方

1. まず、`npx create-next-app`で下ごしらえをします。
   ```sh
   $ npx create-next-app
   √ What is your project named? ... sofken2
   √ Would you like to use TypeScript? ... No / [Yes]
   √ Would you like to use ESLint? ... No / [Yes]
   √ Would you like to use Tailwind CSS? ... No / [Yes]
   √ Would you like your code inside a `src/` directory? ... [No] / Yes
   √ Would you like to use App Router? (recommended) ... No / [Yes]
   √ Would you like to use Turbopack for `next dev`? ... [No] / Yes
   √ Would you like to customize the import alias (`@/*` by default)? ... No / [Yes]
   Creating a new Next.js app in <path-to-your-workspace>/sofken2.
   ```

2. `npm install`で流し込みましょう。
   ```sh
   $ npm install
   up to date, audited 574 packages in 4s

   286 packages are looking for funding
     run `npm fund` for details
   
   found 0 vulnerabilities
   ```

3. 最後に`npm run dev`で下ごしらえは完了です！ここで[味見もしておきましょう](http://localhost:3000)。
   ```sh
   $ npm run dev

   > sofken2@0.1.0 dev
   > next dev
   
      ▲ Next.js 15.1.3
      - Local:        http://localhost:3001
      - Network:      http://192.168.1.x:3001
      - Experiments (use with caution):
        · typedRoutes
   
    ✓ Starting...
    ✓ Ready in 12.8s
   ```

4. レシピはここまでです。あとは是非自らに秘めたる独創性を発揮して、楽しんでね！
